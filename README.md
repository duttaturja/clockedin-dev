## âš™ï¸ Configuration

Open **Settings** (`Ctrl+,`) and search for **CLockedIn**:

| Setting             | Type    | Default | Description                      |
| ------------------- | ------- | ------- | -------------------------------- |
| `clockedin.enabled` | boolean | `true`  | Enable or disable the extension. |
